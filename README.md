Crystal Collector!

This game utilizes html, css, javascript, and jquery to create a number guessing game. The game includes 4 crystals, each set to a random number that is not known to the user, and a random target number. The value of the crystals is only known to the users once they are clicked on. That value will then be added to the user's guess total. If the user's guess equals the random target number then the user wins, if not the loser loses and the game resets.
